The Project

About CyberGuard

A full-stack cybersecurity authentication platform — Diploma Project.

🎯 Project Objective

CyberGuard demonstrates real-world authentication security techniques used in enterprise applications. Built with Python Flask and SQLite (MySQL-compatible), it showcases how modern web applications protect user credentials and sessions against common attacks.

🔧 Technology Stack

  • Python 3.11+ / Flask 3.x
  • SQLite (MySQL-compatible schema)
  • HTML5 + CSS3 + Bootstrap 5
  • Vanilla JavaScript (ES6+)
  • Font Awesome icons

🛡 Security Features

  • Scrypt password hashing
  • Two-Factor Authentication (OTP)
  • Login brute-force protection
  • CSRF token protection
  • Math CAPTCHA on forms
  • Email verification
  • Full audit logging
  • Session timeout
