Email → Fernet AES-128
Emails encrypted with AES-128-CBC + HMAC-SHA256. Only the server with the secret ENCRYPTION_KEY can decrypt. Database admins see only ciphertext.
Email Hash → HMAC-SHA256
A deterministic but irreversible fingerprint of the email. Used for fast WHERE email_hash=? lookups without decrypting every row.
Password → Scrypt KDF
One-way key derivation. Needs 64 MB RAM per guess — GPU brute-force is infeasible. Each user has a unique random salt so identical passwords look different.
