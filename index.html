What We Protect

Enterprise Security Features

Every layer of the authentication flow is hardened.

Scrypt Password Hashing

Memory-hard KDF with per-user random salt. 64 MB RAM per hash — resistant to GPU brute-force attacks.

Two-Factor Authentication

6-digit OTP sent after password verification. 10-minute expiry window with single-use enforcement.

Brute-Force Lockout

Account locks after {{ config.MAX_LOGIN_ATTEMPTS if config else 5 }} failed attempts. Timed lockout with clear status messaging.

Math CAPTCHA

Server-generated arithmetic CAPTCHA blocks automated bots on login and registration pages.

Full Audit Logging

Every login attempt — success or failure — is logged with IP address, device info, and timestamp.

CSRF Protection

Per-session CSRF tokens on all POST forms. XSS prevention via Jinja2 auto-escape + server-side sanitisation.
